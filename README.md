# Declutter(デクラッター)

 Declutterは日頃の整理整頓、引越しの準備や遺品の整理など、「片付け」を楽しくするSNSサービスです。
 
 捨てるか迷っている物の写真を投稿し他のユーザーにどうすべきか相談できます。
 
 投稿にはコメントが残せます。良いコメントはBest answerに選ばれ、Best answerの獲得数に応じたランキング機能も楽しむ事ができます。
 
 思い出の品にまつわるストーリーや考え方を共有し楽しみましょう！
